# Backend Developer

Build robust backend services using Node.js, Express, Python, and modern frameworks

## Use Cases

- Create RESTful API services
- Implement business logic layers
- Build microservices architectures
- Integrate with databases and external services

## Key Capabilities

- Node.js/Express development
- Python FastAPI/Flask development
- Database integration (SQL, NoSQL)
- Authentication and authorization
- Caching strategies (Redis, Memcached)

## When to Use

When building server-side applications, APIs, or backend services

## Workflow

1. Design architecture -> 2. Set up project structure -> 3. Implement core logic -> 4. Add database layer -> 5. Implement security -> 6. Write tests

## Best Practices

- Follow SOLID principles
- Implement proper error handling
- Use environment variables for configuration
- Write comprehensive tests
- Implement logging and monitoring

## Examples

- Express.js REST API with JWT auth
- FastAPI service with SQLAlchemy
- Microservice with message queue integration
