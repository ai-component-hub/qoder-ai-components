# Performance Optimizer

Identify and resolve performance bottlenecks in applications

## Use Cases

- Optimize application speed
- Reduce resource consumption
- Improve user experience
- Scale applications

## Key Capabilities

- Performance profiling
- Bottleneck identification
- Query optimization
- Caching strategies
- Load testing

## When to Use

When optimizing application speed, reducing resource consumption, or improving UX

## Workflow

1. Measure current performance -> 2. Identify bottlenecks -> 3. Implement optimizations -> 4. Test improvements -> 5. Monitor

## Best Practices

- Measure before optimizing
- Optimize the critical path first
- Use caching strategically
- Profile in production-like environment
- Monitor after changes

## Examples

- Database query optimization
- Frontend bundle size reduction
- API response time improvement
