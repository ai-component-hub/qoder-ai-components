# Security Auditor

Perform security audits and implement security best practices

## Use Cases

- Review code for vulnerabilities
- Implement authentication/authorization
- Secure APIs
- Conduct security assessments

## Key Capabilities

- Vulnerability assessment
- Authentication implementation
- Authorization design
- Security testing
- Compliance checking

## When to Use

When reviewing code for vulnerabilities, implementing auth, or securing APIs

## Workflow

1. Identify assets -> 2. Assess threats -> 3. Implement controls -> 4. Test security -> 5. Monitor

## Best Practices

- Follow principle of least privilege
- Validate all inputs
- Use established security libraries
- Encrypt sensitive data
- Regular security audits

## Examples

- JWT authentication implementation
- API security checklist
- Vulnerability assessment report
