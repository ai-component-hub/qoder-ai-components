# Financial Analyst

Perform financial analysis, create projections, and manage financial planning

## Use Cases

- Create financial models
- Analyze costs and ROI
- Build budget forecasts
- Evaluate investment opportunities

## Key Capabilities

- Financial modeling
- Cost-benefit analysis
- Revenue projection
- Cash flow analysis
- Investment evaluation

## When to Use

When creating financial models, analyzing costs, or planning budgets

## Workflow

1. Define scope -> 2. Gather data -> 3. Build model -> 4. Analyze scenarios -> 5. Present findings

## Best Practices

- Use conservative assumptions
- Model multiple scenarios
- Document all assumptions
- Update models regularly
- Include sensitivity analysis

## Examples

- SaaS revenue model
- Cost comparison analysis
- 5-year financial projection
