## What We Did

This repository represents a **curation and organization effort**:

✅ **What we did:**
- Collected components from multiple open-source repos
- Organized them into 15 categories (security, web-frontend, cloud, etc.)
- Converted formats to be Qoder IDE compatible
- Added metadata (name, description, source, license)
- Created category system and auto-loading infrastructure

❌ **What we didn't do:**
- Create the original components (all credit to original authors)
- Claim ownership of others' work
- Modify the core content/functionality of components
- Remove or alter license files or copyright notices

## How to Attribute

When using components from this repository:

1. **For the collection as a whole:**
   - Credit: "Curated collection from qoder-ai-components"
   - License: MIT

2. **For individual components:**
   - Check the component's frontmatter for original source
   - Credit the original author
   - Follow the original component's license

3. **Example attribution:**
   ```
   This project uses the 007 security audit skill from 
   antigravity-awesome-skills by renat (MIT License),
   as part of the qoder-ai-components curated collection.
   ```

## License Compatibility

We've aimed to include components with **permissive licenses** that are compatible with the MIT license:

| License Type | Compatible? | Notes |
|--------------|-------------|-------|
| MIT | ✅ Yes | Fully compatible |
| Apache 2.0 | ✅ Yes | Compatible, includes patent grants |
| BSD 2/3 Clause | ✅ Yes | Fully compatible |
| CC BY 4.0 | ✅ Yes | Compatible for documentation |
| GPL v3 | ⚠️ Review | Viral license - may require special handling |
| No License | ❌ No | All rights reserved - should be removed |
