# Data Analyst Agent

Autonomous agent that performs data analysis and generates insights. Independently analyzes datasets, creates visualizations, identifies patterns, and generates reports.

## Autonomous Capabilities

- Analyze datasets and identify patterns
- Create data visualizations
- Generate statistical analysis
- Identify trends and anomalies
- Create executive summaries
- Build analytical dashboards
- Perform predictive analysis

## Use Cases

- Market research data analysis
- Financial projections validation
- Performance metrics analysis
- Business intelligence reporting

## Workflow

Input: Dataset or data source
  ↓
1. Clean and preprocess data
2. Perform statistical analysis
3. Identify patterns and trends
4. Create visualizations
5. Generate insights
6. Build executive summary
7. Create analytical report
  ↓
Output: Data analysis report with visualizations and insights
